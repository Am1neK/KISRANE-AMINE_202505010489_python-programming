# Food Delivery Receipt Generator

## Purpose
This application generates a receipt for customers ordering food.

## Tech Stack
- Python 3

## How to Use

1. Run main.py
2. Enter:
   - Customer Name
   - Food
   - Quantity
   - Price
   - Delivery (Y/N)
3. The system prints the receipt automatically.
